# Uses

The tools I use day to day. Updated occasionally.

## Editor

I use **Trae** for most of my projects. For **Sui Move**, I stick with **VS Code**, and I use **Zed** specifically for reading and studying codebases. I'm always testing new tools and editors to see what fits my workflow better.

## AI & Agents

I'm deep into the AI ecosystem and love exploring new models and agents. I primarily use **OpenCode** configured with various API keys. My current rotation includes **DeepSeek V4**, and **GLM models**. I also use **Claude Code** for specific tasks. Honestly, I'm always testing everything—if there's a new model or agentic tool out there, I'm probably exploring it.

## Terminal

I use the default **macOS Terminal**. It gets the job done without any extra fuss.

## Languages

**TypeScript**, **Python**, **Go**, and **Sui Move**. I'm also planning to sit down and learn **Rust** very soon.

## Stack (default)

When starting something new: **Vite + React** on the frontend, **Flask** for the backend, and **Postgres** for the database. I deploy everything to a **Coolify** VPS.

## Hardware

My main machine is a **MacBook M4 Pro**. I'm incredibly productive with it because I can work from anywhere—lying down, sitting, or standing. It's fast, quiet, stays cool, and has great battery life.
